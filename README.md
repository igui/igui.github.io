# Ignacio Avas - Portfolio

Personal portfolio website for Ignacio Avas — AI/ML Engineer.

- **Framework:** React 19 + Vite
- **Styling:** Tailwind CSS + Radix UI
- **Animations:** Framer Motion
- **Language:** TypeScript

## Development

```bash
pnpm install
pnpm dev
```
